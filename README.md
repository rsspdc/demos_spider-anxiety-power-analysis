# Arachnophobia Power analysis

> [!NOTE]
> This is an example project for [RSSPDC](https://rsspdc.org)

## Description

This project is to perform a power analysis on some preliminary data about the anxiety levels of arachnophobes when exposed to real tarantulas Vs. images of tarantulas.
We wish to see how many arachnophobes we would need to gather data on to determine if there is a significant different in anxiety levels between these exposures.
## Project Structure

The analysis is located in the `power-analysis.qmd` [Quarto Notebook](https://quatro.org).
The data can be found in the `data` directory
